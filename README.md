# Интернет-магазин светильников Light

## О проекте
Проект создан для проходжения летней практики в компании Диол

## Запуск
Для запуска сайта необходимо
- Клонировать репозиторий (`git clone https://github.com/subreax/mag-svet`)
- Открыть папку src/
- Запустить index.html
